# AI Resume Screening System

## 📌 Overview
This project builds a Resume Screening System that evaluates candidates based on job description.

## 🚀 Features
- Skill Extraction
- Resume Matching
- Scoring System (0–100)
- Explanation for scores

## 🛠️ Technologies Used
- Python
- Jupyter Notebook

## 📊 Output Example
- Strong Candidate → Score: 100
- Average Candidate → Score: 66
- Weak Candidate → Score: 0

- resume_screening_system.ipynb
